# Introduction

A [widget](https://widget.xp.network/?widget=true&wsettings=true#) is a tool that enables codeless integration of the Multi-Chain NFT Bridge to any UI. Every component of the bridge is adjustable to the needs of a user, including the color palette of its components and the chains and wallets available to the user.
